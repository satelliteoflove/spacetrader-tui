# Spaceships & Equipment Specification

## Overview
Spaceships are the player’s primary asset. Each ship type has unique stats and upgrade options. Equipment enhances ship capabilities and survivability.

## Ship Types
- Flea, Gnat, Firefly, Mosquito, Bumblebee, Beetle, Hornet, Grasshopper, Termite, Wasp
- Each ship has:
  - Hull strength
  - Cargo bays
  - Weapon slots
  - Shield slots
  - Gadget slots
  - Crew quarters
  - Max travel range (parsecs per tank)
  - Price

## Buying/Selling Ships
- Ships bought/sold at shipyards
- Trade-in value includes current cargo/equipment
- Cargo not wanted by local market is lost on sale
- Escape pod and insurance transfer to new ship

## Equipment
- Weapons: Pulse, Beam, Military lasers (damage, cost, tech level required)
- Shields: Energy, Reflective (protection, recharge, tech level required)
- Gadgets: Extra cargo bays, Navigation, Targeting, Auto-repair, Cloaking
  - Each gadget provides unique bonuses (e.g., cloaking = stealth, navigation = piloting bonus)
- Escape pod: Automatic ejection on ship destruction
- Insurance: Refunds ship value if escape pod used (daily premium, no-claim bonus)

## Upgrades
- Slots limit number of each equipment type
- Equipment purchased at shipyards (availability depends on system tech level)

## Damage & Repairs
- Hull/shield damage from combat; shields recharge over time
- Repairs at shipyards (cost based on damage/ship size)

## Persistent State
- Ship stats, equipment, and upgrades persist across saves/loads
